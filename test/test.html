How to Test

  1. Open chrome://extensions in Chrome
  2. Enable "Developer mode" (top right toggle)
  3. Click "Load unpacked" and select the extension folder
  4. Return to this page and right-click on any captcha image
  5. Select "Read Captcha" from the context menu
  6. The numbers will be automatically copied to your clipboard!
